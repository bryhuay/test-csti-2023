<h1>USER-MICROSERVICE</h1>
<h3>Jerson Huayta</h3>

## Description

A microservice that allows you to validate the existence of a user in a local DB, as well as verifies credentials to generate an access token. It also has an endpoint that allows the creation of users in the SQLite database

## Technologies

<ul>
  <li>NestJs</li>
  <li>Sqlite</li>
  <li>Docker</li>
  <li>JWT</li>
</ul>


## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
